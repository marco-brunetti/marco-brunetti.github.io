Crosscountry Racer (2020)

Although I initially intended to create an infinite runner, after some prototyping I settled with an infinite racer instead. This game features car color modification capabilities, an Android build, and a “radio” system for music listening while driving. The car accelerates as time passes, and you earn coins when you pass other cars, with a bonus for near misses. This game helped me understand the importance of learning good code practices, as the code became so unreadable that I had to discontinue the project after a few months.

Breakthroughs:

  • Materials: I learned how to modify materials in-game and used this knowledge for the car color modification logic.
  • Optimization: I started using the profiler and breakpoints to find bottlenecks and bugs in the code, which greatly helped in future games.
  • 3D models: I learned how to reduce the vertex count in most models in the game, facilitating the infinite racer mechanic.
  • Android build: The game was optimized enough to run on mid-tier Android phones.

Drawbacks:

  • Unmaintainable code: The code was poorly structured and had many convoluted fixes, forcing me to discontinue the project.
