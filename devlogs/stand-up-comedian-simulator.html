Stand Up: Comedian Simulator (2024)

This game was created for the Global Game Jam and developed in 10 days. The theme of this game jam was “Make Me Laugh.” At the time, I had a hard time managing events in C#, so I decided to create the game entirely with events and coroutines, with no updates (ticks) except for input detection. I created a first-person stand-up comedian simulator with various jokes divided into good or bad ones. The player doesn't know which jokes belong to each group, so they must use their intuition to pick the good jokes for the audience. Aside from the coding, I wanted a game that feels polished, so I needed to save time by cutting down on any unnecessary mechanics. The game is in first-person perspective, but the character does not move in the scene; they can only look around to select jokes and tell them to the audience.

Breakthroughs:

  • Events: I improved my event management skills and learned a lot about Unity's coroutines. This helped me at work as I started learning about real asynchronous code shortly after.
  • Polish: I achieved a relatively polished game in a short amount of time by cutting anything unnecessary to the main mechanic.
  • Tools: I created some tools, such as one that converts JSON text into the UI and gameplay text elements.
