Reflection from the Abyss (2023)

This game was made in 72 hours for the IGDA PY Game Jam. For this project, I teamed up with a 2D and a 3D artist to create an abstract escape room inspired by Narcissus, a character from Greek mythology. We needed to develop modular code rapidly so that we didn't have to create many interactable objects. We also needed some kind of inventory system. Although this wasn't my first team jam, it helped me improve my communication and planning with artists.

Breakthroughs:

  • Inventory: I created a simple but effective inventory system rapidly, thanks to the lessons learned from creating Cursed Silence.
  • Art: I learned how to combine 2D and 3D art in the same scene.
  • Mirrors: I learned how to create mirrors using texturing and cameras.
  • Modular code: It was the first jam where I tested how to create modular and generic code rapidly.
